# 🏦 pgAdminPerbankan-PHPweb

Aplikasi web sederhana menggunakan **PHP** yang terhubung ke **PostgreSQL (pgAdmin)** untuk mengelola data perbankan.  
Fitur utama berupa **CRUD (Create, Read, Update, Delete)** pada beberapa tabel:

- Nasabah
- Cabang Bank
- Rekening
- Transaksi
- Relasi Nasabah–Rekening

## ⚙️ Teknologi
- PHP (Native)
- PostgreSQL (pgAdmin sebagai DBMS GUI)
- HTML + CSS (Dashboard Layout)
- XAMPP (Apache + PHP)
